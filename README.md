# react-native-en-bot-sdk

react native plugin for ENBotSDK

## Installation

```sh
npm install react-native-en-bot-sdk
```

## Usage

```js
import { multiply } from "react-native-en-bot-sdk";

// ...

const result = await multiply(3, 7);
```
